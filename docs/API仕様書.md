## 3. 詳細仕様
### 例：クライアント一覧取得
- メソッド: GET
- パス: /api/clients
- 認証: 必須
- リクエストパラメータ: なし
- レスポンス例:
```json
[
  { "id": 1, "name": "株式会社サンプル" }
]
```
- エラーコード:
  - 401: 認証エラー
  - 500: サーバーエラー
