# 機能書：クライアント管理

## 概要
クライアント情報（企業名、担当者、連絡先等）の登録・一覧・詳細・編集・削除を行う機能。

## 機能一覧
- クライアント一覧表示
- クライアント新規登録
- クライアント詳細表示
- クライアント情報編集
- クライアント削除

## 画面・操作
- 一覧画面：登録済みクライアントのリスト表示、詳細・編集・削除ボタン
- 登録画面：新規クライアント情報入力フォーム
- 詳細画面：クライアント情報の表示
- 編集画面：既存クライアント情報の編集

## バリデーション
- 必須項目：企業名、担当者名、連絡先
- 形式チェック：メールアドレス等

## 権限
- ログインユーザーのみ操作可能

## 備考
- 削除時は論理削除または確認ダイアログ表示
