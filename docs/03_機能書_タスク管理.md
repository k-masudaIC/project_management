# 機能書：タスク管理

## 概要
プロジェクトに紐づくタスクの登録・一覧・詳細・編集・削除、進捗・状態管理を行う機能。

## 機能一覧
- タスク一覧表示
- タスク新規登録
- タスク詳細表示
- タスク情報編集
- タスク削除
- プロジェクト別絞り込み
- ステータス別絞り込み

## 画面・操作
- 一覧画面：タスクリスト、絞り込み、詳細・編集・削除ボタン
- 登録画面：新規タスク情報入力フォーム
- 詳細画面：タスク情報の表示
- 編集画面：既存タスク情報の編集

## バリデーション
- 必須項目：タスク名、プロジェクト、担当者
- 形式チェック：日付等

## 権限
- ログインユーザーのみ操作可能

## 備考
- 削除時は論理削除または確認ダイアログ表示
