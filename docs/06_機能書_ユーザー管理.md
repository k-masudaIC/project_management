# 機能書：ユーザー管理

## 概要
システム利用ユーザーの登録・一覧・詳細・編集・削除、権限設定、プロフィール管理を行う機能。

## 機能一覧
- ユーザー一覧表示
- ユーザー新規登録
- ユーザー詳細表示
- ユーザー情報編集
- ユーザー削除
- 権限（ロール）設定
- プロフィール画像アップロード
- パスワードリセット

## 画面・操作
- 一覧画面：ユーザーリスト、詳細・編集・削除ボタン
- 登録画面：新規ユーザー情報入力フォーム
- 詳細画面：ユーザー情報の表示
- 編集画面：既存ユーザー情報の編集

## バリデーション
- 必須項目：氏名、メールアドレス、ロール
- メールアドレス形式チェック
- パスワード強度チェック

## 権限
- 管理者のみ全ユーザー操作可能
- 一般ユーザーは自身のプロフィールのみ編集可

## 備考
- パスワードはハッシュ化して保存
- 削除時は論理削除または確認ダイアログ表示
